DeIDGuard – Icon Reference
==========================

All three icons (icon16.png, icon48.png, icon128.png) are programmatically
generated teal shield icons. They are production-ready PNG files with
transparent backgrounds and do not require replacement to load the extension.

Visual Description
------------------
Shape:   Classic heraldic shield — wide at the top, tapering to a point
         at the bottom center.

Colors:
  Primary fill:  Teal (#0ea5a0) — bright upper portion
  Dark fill:     Dark teal (#0b6e6b) — lower and shadow portions
  Detail:        White (#ffffff) — interior icon

Interior detail (48px and 128px):
  A simplified padlock icon rendered in white:
  - Circular arc at top (the shackle)
  - Rounded rectangle body below
  - Small teal keyhole dot in the center of the body

Interior detail (16px):
  A simple white cross/plus shape for maximum clarity at small sizes.

Symbolism:
  Shield = protection / security
  Lock   = privacy / local-only processing
  Teal   = trustworthy, health/tech aesthetic

To Replace Icons
----------------
If you want to supply custom icons for Chrome Web Store submission:
  1. Create PNG files at exactly 16x16, 48x48, and 128x128 pixels.
  2. Use a transparent background (PNG-24 or PNG-32 with alpha channel).
  3. Ensure the 128x128 version meets the Chrome Web Store promotional
     icon guidelines (full color, no rounded corners — Chrome adds them).
  4. Replace the three files in this /icons/ folder.
  5. Reload the extension in chrome://extensions.

Chrome Web Store Additional Assets (not required for local use)
---------------------------------------------------------------

  - Promotional tile: 920x680 px (large) — optional
  - Screenshots: at least 1280x800 or 640x400 px
